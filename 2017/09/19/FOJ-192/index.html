FOJ 192. Fences

Muller Lee
2017-09-19
Algorithm
FOJ

競程作業，2017 TOPC Problem E：FOJ-192

題目是要你算出大圓面積跟裡面的小多邊形面積的差
大圓直接算就好了
小多邊形的面積有點麻煩，要拿已知的條件去二分搜，猜出他的外接圓半徑
高級題解在這：https://hackmd.io/s/Hy6LFym_b

這題我超雷，因為 C 的 abs 是專屬整數用的
浮點數要用 fabs
但是 C++ 是通用
結果我自己跑都很正常但是送出去一直 WA
